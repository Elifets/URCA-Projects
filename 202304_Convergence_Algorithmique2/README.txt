PROJET MA0844 - ERTAS Elif 
Système d'exploitation : WINDOWS 

------------- DESCRIPTION DES FICHIERS ------------------

solveurLU.c --> code en langage C (norme ANSI-C90) qui permet de résoudre un système linéaire tridiagonal (application sur un exemple) 
solveurMDF.c --> code en langage C (norme ANSI-C90) qui permet de résoudre un problème elliptique 1D et MDF 
main.py --> code en Python3 qui réalise le Pré- et le Post-traitement et qui appel le programme solveurMDF pour la réalisation du traitement 
	
NB : dans le fichier main.py décommenter la ligne 117 pour l'affichage dans la console sous LINUX 

------------- MISE EN OEUVRE -----------------------

1) Compiler et exécuter le programme solveurMDF.c pour la création des fichiers .txt nécessaire dans le programme main.py 
2) Lancer le programme main.py et répondre aux dialogues dans la console, l'exécution du fichier solveurMDF.exe se fait automatiquement et la suite du Post-Traitement également. 

/!\ Si le programme main.py ne s'éxecute pas correctement (erreurs ou autre) le ré-exécuter à nouveau sans rien modifier jusqu'à que ça fonctionne /!\
